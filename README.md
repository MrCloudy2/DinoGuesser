# 🦕 DinoGuess — Pipeline Reference

## Install dependencies (once)

```bash
pip install libzim beautifulsoup4 lxml requests --break-system-packages
```

---

## Pipeline (run in order from the pipeline/ directory)

```
Step 1  1_fetch_list.py               ~5 sec    one Wikipedia API call, gets ~1400 names
Step 2  2_extract_pages_from_zim.py   ~5 min    article text from ZIM (for Qwen hints)
Step 3  3_extract_from_zim.py         ~10 min   images from ZIM
Step 3b 3b_generate_picker.py         instant   builds picker.html
Step 3c serve_picker.py               ongoing   serve picker, tag Art / Fossil in browser
Step 4  4_extract_hints.py            ~60 min   Qwen generates 5 hints per dino
Step 5  6_flag_validity.py            ~10 sec   flags dubious / synonym / disputed genera
Step 6  5_build_db.py                 ~30 sec   assembles www/dinosaurs.json
```

### Step 1 — Fetch dinosaur name list
```bash
python 1_fetch_list.py
# Output: ../data/dinosaurs_list.txt
```

### Step 2 — Extract article text from ZIM
```bash
python 2_extract_pages_from_zim.py --zim /path/to/wikipedia_en_all_maxi_*.zim
# Output: ../data/raw_pages/{name}.txt  (safe to interrupt, resumes)
```

### Step 3 — Extract images from ZIM
```bash
python 3_extract_from_zim.py --zim /path/to/wikipedia_en_all_maxi_*.zim
# Output: ../data/images/{name}_1.jpg etc, ../data/image_lists/{name}.json
# --limit N   max images per dino (default 5)
```

### Step 3b/3c — Tag images as Art / Fossil
```bash
python 3b_generate_picker.py   # builds ../picker.html
python serve_picker.py         # opens http://localhost:8001/picker.html
# In browser: click images to tag as 🎨 Art or 🦴 Fossil
# Click "Save selections.json" → place it at ../data/selections.json
```
Images tagged as Art become the main game image.
Images tagged as Fossil appear via the 🦴 toggle on the image card.

### Step 4 — Generate hints with Qwen (Ollama)
```bash
# First: check your model name
ollama list
# Then set MODEL_NAME in 4_extract_hints.py (top of file)

python 4_extract_hints.py
# Output: ../data/hints_cache/{name}.json  (safe to interrupt, resumes)
# Estimated time: ~45-70 min for 1400 dinos on Qwen 35B
```

Hints are ordered **hardest → easiest** (index 0 = geological/stratigraphic detail,
index 4 = most famous obvious fact).

### Step 5 — Flag validity
```bash
python 6_flag_validity.py
# Output: ../data/validity.json
# Prints summary: how many valid / uncertain / disputed / excluded
```

Statuses:
- `valid`        → always in game
- `disputed`     → only with 🔬 Rare toggle ON
- `uncertain`    → only with 🔬 Rare toggle ON
- `nomen_dubium` → never in game (named from one tooth, indistinguishable etc.)
- `synonym`      → never in game (junior synonym of another genus)

### Step 6 — Assemble final database
```bash
python 5_build_db.py
# Output: ../www/dinosaurs.json + copies images to ../www/images/
```

## Game modes

| Mode | Description |
|---|---|
| Daily 📅 | Same dino for everyone, resets UTC midnight |
| Random 🎲 | New random dino each game |
| Hard 💀 | Image hidden until game ends, start with 1 hint |
| Rare 🔬 | Include disputed / uncertain genera |

Up to 5 guesses. Each wrong guess reveals the next hint (hardest → easiest).
Art / Fossil toggle on image card when both images exist.
